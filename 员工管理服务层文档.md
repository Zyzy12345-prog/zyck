# 员工管理服务层文档

## 📋 概述

员工管理服务层（EmployeeService）封装了所有员工管理相关的业务逻辑，提供可复用的服务方法。

**文件位置**：`services/employeeService.js`  
**设计模式**：单例模式  
**创建时间**：2026-01-23

---

## 🏗️ 架构设计

### 服务层职责
```
Controller（控制器）
    ↓ 调用
Service（服务层）← 业务逻辑
    ↓ 调用
Model（模型层）← 数据操作
```

### 优势
- ✅ **业务逻辑分离** - 控制器只负责请求响应
- ✅ **代码复用** - 服务方法可在多处调用
- ✅ **易于测试** - 业务逻辑独立测试
- ✅ **事务管理** - 统一的事务处理
- ✅ **错误处理** - 集中的错误处理

## 💻 详细API文档

### 1. 账号管理

#### 1.1 createEmployee(data)

创建员工账号，自动生成初始密码

**参数**：
```javascript
{
  employeeId: 'EMP001',      // 必填：员工工号
  username: 'zhangsan',      // 必填：用户名
  password: 'password123',   // 可选：密码（不提供则自动生成）
  fullName: '张三',          // 必填：姓名
  email: 'zhangsan@example.com', // 必填：邮箱
  phone: '13800138000',      // 可选：电话
  department: '销售部',      // 可选：部门
  position: '销售经理',      // 可选：职位
  avatar: 'url',             // 可选：头像URL
  hireDate: '2024-01-15',    // 可选：入职日期
  roleId: 2,                 // 可选：角色ID
  permissions: [],           // 可选：权限数组
  accessLevel: 5             // 可选：访问等级
}
```

**返回值**：
```javascript
{
  employee: Employee对象,
  initialPassword: '自动生成的密码' // 仅在未提供密码时返回
}
```

**使用示例**：
```javascript
const employeeService = require('../services/employeeService');

// 创建员工（自动生成密码）
const result = await employeeService.createEmployee({
  employeeId: 'EMP001',
  username: 'zhangsan',
  fullName: '张三',
  email: 'zhangsan@example.com',
  department: '销售部',
  roleId: 5
});

console.log('初始密码:', result.initialPassword);
console.log('员工信息:', result.employee);
```

**特性**：
- ✅ 自动生成8位随机密码
- ✅ 验证工号、用户名、邮箱唯一性
- ✅ 验证角色是否存在
- ✅ 使用事务保证数据一致性
- ✅ 密码自动加密

#### 1.4 bulkCreateEmployees(employeesData)

批量创建员工

```javascript
{
```

```javascript
  ],
  failed: [
    {
      employeeId: 'EMP002',
      error: '错误信息'
    }
  ]
}
```

**使用示例**：
```javascript
const result = await employeeService.bulkCreateEmployees(employeesData);
console.log(`成功: ${result.success.length}, 失败: ${result.failed.length}`);
```

### 2. 权限管理

#### 2.1 assignRole(employeeId, roleId)

为员工分配角色，并同步角色权限

**参数**：
- `employeeId` - 员工ID
- `roleId` - 角色ID

**返回值**：Employee对象

**使用示例**：
```javascript
const employee = await employeeService.assignRole(1, 2);
console.log('新角色:', employee.role.name);
console.log('权限:', employee.permissions);
```

**特性**：
- ✅ 验证员工和角色存在性
- ✅ 自动同步角色权限到员工
- ✅ 使用事务

---

#### 2.2 updatePermissions(employeeId, permissions)

更新员工个人权限

**参数**：
- `employeeId` - 员工ID
- `permissions` - 权限代码数组

**使用示例**：
```javascript
const employee = await employeeService.updatePermissions(1, [
  'client:read',
  'client:write',
  'call:read',
  'call:create'
]);
```

**特性**：
- ✅ 验证权限代码有效性
- ✅ 使用事务

---

#### 2.3 checkPermission(employeeId, permissionCode)

检查员工是否拥有特定权限

**参数**：
- `employeeId` - 员工ID
- `permissionCode` - 权限代码

**返回值**：Boolean

**使用示例**：
```javascript
const hasPermission = await employeeService.checkPermission(1, 'client:write');
if (hasPermission) {
  // 允许操作
}
```

**检查顺序**：
1. 超级权限（`*`）
2. 角色权限
3. 个人权限

---

#### 2.4 bulkAssignRole(employeeIds, roleId)

批量分配角色

**参数**：
- `employeeIds` - 员工ID数组

**返回值**：
```javascript
{
  success: [1, 2, 3],
  failed: [

  ]
}
```

**使用示例**：
```javascript
const result = await employeeService.bulkAssignRole([1, 2, 3, 4], 5);
```

### generateInitialPassword()

生成8位随机密码

**返回值**：String

**特点**：
- 包含大小写字母和数字
- 排除易混淆字符（0、O、1、I、l等）
- 长度固定8位

获取日期范围

**参数**：
- `period` - 'today'/'week'/'month'/'all'

**返回值**：
```javascript
{
  start: Date对象,
  end: Date对象
}
```

---

### calculateWorkloadLevel(todayCalls, weekCalls, pendingClients)

计算工作量等级

**返回值**：'low'/'medium'/'high'/'very_high'

**计算公式**：
```
score = todayCalls * 2 + weekCalls + pendingClients * 0.5

score < 20: low
score < 50: medium
score < 80: high
score >= 80: very_high
```

## 💡 使用示例

### 示例1：创建员工并分配角色

```javascript
const employeeService = require('../services/employeeService');

// 1. 创建员工
const result = await employeeService.createEmployee({
  employeeId: 'EMP001',
  username: 'zhangsan',
  fullName: '张三',
  email: 'zhangsan@example.com',
  department: '销售部',
  position: '销售专员'
});

console.log('初始密码:', result.initialPassword);

// 2. 分配角色
const employee = await employeeService.assignRole(
  result.employee.id,
  5  // 销售员角色
);

console.log('角色:', employee.role.name);
console.log('权限:', employee.permissions);
```

### 示例2：获取员工工作统计

```javascript
// 获取本月统计
const stats = await employeeService.getEmployeeStats(1, 'month');

console.log('客户总数:', stats.clients.total);
console.log('通话总数:', stats.calls.total);
console.log('成功率:', stats.calls.successRate + '%');

// 获取工作量
const workload = await employeeService.getEmployeeWorkload(1);
console.log('今日通话:', workload.calls.today);
console.log('工作量等级:', workload.workload.level);
console.log('建议:', workload.workload.recommendation);
```

### 示例3：批量操作

```javascript
// 批量创建员工
```
