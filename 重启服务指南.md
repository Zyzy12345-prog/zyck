## ⚠️ 如果还有问题

### 查看后端日志
后端终端会显示详细的错误信息，包括：
```
获取客户详情失败: [错误详情]
```

### 查看前端控制台
按 F12 打开浏览器控制台，查看具体的错误信息。

### 常见错误

**错误1: "客户不存在"**
- 原因：数据库中没有该客户
- 解决：返回客户列表，点击实际存在的客户

**错误2: 关联模型错误**
- 原因：数据库关联配置问题
- 解决：已在本次修复中解决

**错误3: 500 Internal Server Error**
- 原因：后端代码错误
- 解决：查看后端终端的错误日志
