## 💻 常用代码

### 创建记录
```javascript
const call = await CallRecord.create({
  clientId: 1,
  userId: 1,
  phoneNumber: '13800138000',
  callType: 'manual',
  direction: 'outbound',
  status: 'completed',
  duration: 300,
  summary: '客户感兴趣',
  tags: ['意向客户'],
  satisfaction: 5
});
```

### 查询记录
```javascript
const calls = await CallRecord.findAll({
  include: [
    { model: Client, as: 'Client' },
    { model: User, as: 'User' }
  ],
  where: { status: 'completed' },
  order: [['callTime', 'DESC']]
});
```

### 获取统计
```javascript
const stats = await CallRecord.getStatistics({
  userId: 1,
  startDate: '2026-01-01',
  endDate: '2026-01-31'
});
```

## 🔍 枚举值速查

### call_type
- `manual` - 手动拨打
- `auto` - 自动外呼
- `callback` - 回拨
- `inbound` - 呼入

### direction
- `inbound` - 呼入
- `outbound` - 呼出

### status
- `initiated` - 已发起
- `ringing` - 振铃中
- `answered` - 已接听
- `completed` - 已完成
- `no_answer` - 未接听
- `busy` - 忙线
- `failed` - 失败

### provider
- `virtual` - 虚拟测试
- `twilio` - Twilio
- `aliyun` - 阿里云
- `tencent` - 腾讯云
- `other` - 其他
