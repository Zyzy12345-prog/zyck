### 阶段二：创建和编辑功能 🚧

#### 3. 创建外呼记录表单组件

**文件：** `client/src/components/CallForm.jsx`

**需要实现的功能：**
```javascript
- 客户选择（下拉框，支持搜索）
- 呼叫类型选择（呼入/呼出）
- 呼叫时间选择（日期时间选择器）
- 通话时长输入（分钟和秒）
- 呼叫结果选择（已接听/未接听/忙线/失败）
- 通话摘要输入（多行文本框）
- 录音文件上传（可选）
- 表单验证
- 提交和取消按钮
```

**开发步骤：**

1. **创建表单组件**
```bash
# 创建文件
client/src/components/CallForm.jsx
client/src/components/CallForm.css
```

2. **实现客户选择功能**
```javascript
// 获取客户列表
const [clients, setClients] = useState([]);
useEffect(() => {
  fetchClients();
}, []);
```

3. **实现表单验证**
```javascript
// 验证必填字段
- 客户必选
- 呼叫类型必选
- 呼叫时间必填
- 呼叫结果必选
```

4. **集成到 Calls.jsx**
```javascript
// 在模态框中使用 CallForm 组件
{showCreateModal && (
  <CallForm
    onSuccess={() => {
      setShowCreateModal(false);
      fetchCalls();
    }}
    onCancel={() => setShowCreateModal(false)}
  />
)}
```

## 🛠️ 技术实现要点

### 1. 客户选择组件
```javascript
// 使用 Select 组件，支持搜索
<select className="input">
  {clients.map(client => (
    <option key={client.id} value={client.id}>
      {client.companyName}
    </option>
  ))}
</select>
```

### 2. 日期时间选择
```javascript
// 使用 HTML5 datetime-local
<input
  type="datetime-local"
  value={callTime}
  onChange={(e) => setCallTime(e.target.value)}
  className="input"
/>
```

### 3. 时长输入
```javascript
// 分钟和秒分开输入
<div className="duration-input">
  <input type="number" placeholder="分钟" />
  <span>:</span>
  <input type="number" placeholder="秒" />
</div>
```

### 4. 文件上传
```javascript
// 使用 FormData 上传录音
const formData = new FormData();
formData.append('recording', file);
await callAPI.uploadRecording(callId, formData);
```

## 💡 开发建议

### 1. 代码复用
- 客户管理和外呼记录的列表页面结构相似，可以复用样式
- 表单组件可以同时用于创建和编辑

### 2. 用户体验
- 添加加载状态
- 添加成功/失败提示
- 表单验证要友好
- 操作要有确认提示

### 3. 性能优化
- 列表数据使用分页
- 图表数据按需加载
- 录音文件延迟加载

### 4. 错误处理
- API 请求失败要有提示
- 表单验证要清晰
- 网络错误要友好提示
