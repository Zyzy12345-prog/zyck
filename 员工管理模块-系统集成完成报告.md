## 🎯 使用指南

### 1. 访问员工管理模块

**步骤**:
1. 使用管理员或经理账号登录系统
2. 在左侧导航栏找到"员工管理"菜单（带有"新"标识）
3. 点击进入员工列表页面

**权限要求**:
- 管理员：完全访问
- 经理：查看和编辑
- 普通员工：无法访问

### 2. 使用数据同步服务

**员工离职处理**:
```javascript
import syncService from '@/services/syncService';

// 在员工列表页面的删除操作中
const handleEmployeeResign = async (employeeId) => {
  try {
    // 调用离职处理服务
    const result = await syncService.handleEmployeeResignation(employeeId, {
      autoTransfer: true,  // 自动转移客户
      notifyManager: true  // 通知部门经理
    });
    
    if (result.success) {
      message.success(`离职处理完成，已转移 ${result.data.clientsTransferred} 个客户`);
      // 刷新列表
      fetchEmployees();
    }
  } catch (error) {
    message.error('离职处理失败：' + error.message);
  }
};
```

**部门删除检查**:
```javascript
import syncService from '@/services/syncService';

// 在部门管理页面的删除操作中
const handleDepartmentDelete = async (department) => {
  try {
    // 检查部门是否可删除
    const checkResult = await syncService.checkDepartmentDeletable(department);
    
    if (!checkResult.deletable) {
      Modal.confirm({
        title: '部门下有员工',
        content: checkResult.message,
        okText: '转移员工',
        onOk: async () => {
          // 打开员工转移对话框
          showTransferDialog(department);
        }
      });
      return;
    }
    
    // 可以删除
    await deleteDepartment(department);
  } catch (error) {
    message.error('检查失败：' + error.message);
  }
};
```
