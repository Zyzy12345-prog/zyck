## 客户线索管理

### 什么是客户线索？

客户线索是指尚未转化为正式客户的潜在客户信息。通过线索管理，销售人员可以：
- 记录潜在客户信息
- 跟踪沟通进度
- 评估线索质量
- 最终转化为正式客户

### 如何使用

#### 1️⃣ 查看线索列表

**路径**: 侧边栏 → 客户线索

**功能**:
- 查看所有线索
- 按状态、优先级、来源筛选
- 搜索线索（公司名、联系人、电话）
- 查看线索统计数据

**线索状态说明**:
- 🆕 **新线索**: 刚创建，尚未联系
- 📞 **已联系**: 已经进行过初步沟通
- ✅ **已确认**: 确认有合作意向
- 💬 **洽谈中**: 正在深入洽谈
- 🎉 **已转化**: 已转化为正式客户
- ❌ **已丢失**: 确认无法合作

#### 2️⃣ 创建新线索

**步骤**:
1. 点击"新增线索"按钮
2. 填写必填信息：
   - 公司名称 ✅
   - 联系人
   - 电话
   - 邮箱
3. 选择线索来源（网站、推荐、电话营销等）
4. 设置优先级（低、中、高、紧急）
5. 点击"确定"保存

**小贴士**:
- 📌 优先级越高，在列表中越靠前
- 📌 建议填写完整信息，便于后续跟进
- 📌 可以在备注中记录重要信息

#### 3️⃣ 分配线索

**单个分配**:
1. 在线索列表中找到目标线索
2. 点击"分配"按钮
3. 选择负责人
4. 确认分配

**批量分配**:
1. 勾选多个线索
2. 点击"批量分配"
3. 选择负责人
4. 确认分配

**权限说明**:
- 👤 销售人员：只能查看分配给自己的线索
- 👥 经理/管理员：可以查看和分配所有线索

#### 4️⃣ 更新线索状态

**方式一：单个更新**
1. 点击线索进入详情页
2. 点击"编辑"按钮
3. 修改状态
4. 保存

**方式二：批量更新**
1. 勾选多个线索
2. 点击"批量更新状态"
3. 选择新状态
4. 确认更新

#### 5️⃣ 转化为客户

当线索确认合作后，可以将其转化为正式客户：

**步骤**:
1. 进入线索详情页
2. 点击"转化为客户"按钮
3. 系统自动创建客户记录
4. 线索状态变更为"已转化"

**注意事项**:
- ⚠️ 转化后无法撤销
- ⚠️ 客户信息会自动从线索复制
- ⚠️ 可以在客户管理中继续完善信息

## 客户公海池

### 什么是客户公海池？

客户公海池是一个共享的客户资源池，用于：
- 存放暂时无人跟进的客户
- 共享优质客户资源
- 提高客户资源利用率
- 避免客户资源浪费

### 客户进入公海池的场景

1. **长期未跟进**: 销售人员N天未跟进的客户
2. **主动释放**: 销售人员主动放弃的客户
3. **离职交接**: 员工离职后的客户
4. **新客户**: 新导入的未分配客户

### 如何使用

#### 1️⃣ 查看公海池

**路径**: 侧边栏 → 客户公海池

**功能**:
- 查看所有可认领的客户
- 按优先级排序
- 查看客户进入原因
- 查看统计数据

**客户状态**:
- 🟢 **可认领**: 可以立即认领
- 🔵 **已认领**: 已被其他人认领
- 🔒 **已锁定**: 暂时不可认领

#### 2️⃣ 认领客户

**单个认领**:
1. 在公海池列表中找到目标客户
2. 点击"认领"按钮
3. 确认认领
4. 客户自动分配给你

**批量认领**:
1. 勾选多个客户
2. 点击"批量认领"
3. 确认认领

**认领规则**:
- 📌 每人每天最多认领X个客户（可配置）
- 📌 认领后需在N天内完成首次跟进
- 📌 长期不跟进会自动回收到公海池

#### 3️⃣ 添加客户到公海池

**适用场景**:
- 客户暂时无法跟进
- 客户不在自己的业务范围
- 希望其他同事接手

**步骤**:
1. 在客户详情页
2. 点击"放入公海池"
3. 填写原因
4. 设置优先级
5. 确认提交

#### 4️⃣ 调整优先级

**权限**: 经理、管理员

**步骤**:
1. 在公海池列表中找到客户
2. 点击"调整优先级"
3. 设置新的优先级（1-10）
4. 保存

**优先级说明**:
- 🔴 8-10: 高优先级（优质客户）
- 🟡 5-7: 中优先级（普通客户）
- 🟢 1-4: 低优先级（待评估客户）

## 转化漏斗分析

### 什么是转化漏斗？

转化漏斗是一个可视化工具，展示客户从线索到成交的完整流程，帮助你：
- 了解转化率
- 发现转化瓶颈
- 优化销售流程
- 提升业绩

### 如何使用

#### 1️⃣ 查看转化漏斗

**路径**: 侧边栏 → 转化漏斗

**页面内容**:
- 📊 转化漏斗图
- 📈 转化趋势图
- 📋 各阶段详情
- 🎯 优先级分布

#### 2️⃣ 筛选数据

**时间筛选**:
1. 点击日期选择器
2. 选择开始和结束日期
3. 自动刷新数据

**常用时间段**:
- 最近7天
- 最近30天
- 本月
- 上月
- 自定义

#### 3️⃣ 理解数据

**转化漏斗图**:
```
新线索 (100)
    ↓
已联系 (80)  - 流失20%
    ↓
已确认 (60)  - 流失25%
    ↓
洽谈中 (40)  - 流失33%
    ↓
已转化 (20)  - 转化率20%
```

**关键指标**:
- **总线索数**: 时间段内的所有线索
- **已转化**: 成功转化为客户的数量
- **转化率**: 已转化 / 总线索数
- **进行中**: 正在跟进的线索数量

**各阶段详情**:
- 查看每个阶段的具体数量
- 识别转化瓶颈
- 优化薄弱环节

**优先级分布**:
- 紧急: 需要立即处理
- 高: 优先处理
- 中: 正常处理
- 低: 有空时处理

#### 4️⃣ 导出报表

**步骤**:
1. 设置筛选条件
2. 点击"导出报表"按钮
3. 选择导出格式（Excel/PDF）
4. 下载文件

**报表内容**:
- 转化漏斗数据
- 各阶段统计
- 转化率分析
- 趋势图表

## 常见问题

### Q1: 线索和客户有什么区别？

**A**: 
- **线索**: 潜在客户，尚未确认合作
- **客户**: 已确认合作的正式客户

线索转化后会自动成为客户。

### Q2: 如何提高线索转化率？

**A**: 
1. 及时跟进新线索
2. 准确评估线索质量
3. 优先处理高优先级线索
4. 记录详细的跟进信息
5. 定期分析转化数据

### Q3: 公海池的客户会自动回收吗？

**A**: 
是的，以下情况会自动回收：
- 销售人员N天未跟进（可配置）
- 销售人员主动释放
- 员工离职

### Q4: 可以撤销线索转化吗？

**A**: 
不可以。线索转化为客户后无法撤销，请谨慎操作。如果误操作，可以：
1. 在客户管理中删除该客户
2. 重新创建线索

### Q5: 批量操作有数量限制吗？

**A**: 
- 批量分配: 最多100条
- 批量更新: 最多100条
- 批量认领: 最多50条

### Q6: 如何查看我的线索统计？

**A**: 
1. 进入"客户线索"页面
2. 顶部显示统计卡片
3. 或进入"转化漏斗"查看详细分析

### Q7: 线索评分是如何计算的？

**A**: 
线索评分基于多个维度：
- 公司规模（30%）
- 预估价值（25%）
- 联系频率（20%）
- 响应速度（15%）
- 行业匹配度（10%）

总分0-100分，分数越高质量越好。

### Q8: 可以导入线索吗？

**A**: 
目前支持手动创建，批量导入功能正在开发中。

### Q9: 线索可以分配给多人吗？

**A**: 
不可以。每个线索只能分配给一个人，避免重复跟进。

### Q10: 如何设置公海池规则？

**A**: 
管理员可以在"系统设置"中配置：
- 自动回收天数
- 每日认领上限
- 优先级规则
- 锁定规则

## 💡 最佳实践

### 线索管理

1. **及时录入**: 获得线索后立即录入系统
2. **完整信息**: 尽可能填写完整的客户信息
3. **准确分类**: 正确设置来源、优先级、状态
4. **定期跟进**: 按计划跟进，避免遗漏
5. **详细记录**: 记录每次沟通的详细内容

### 公海池管理

1. **合理认领**: 根据自己的能力认领客户
2. **及时跟进**: 认领后尽快联系客户
3. **主动释放**: 无法跟进的客户及时放回公海池
4. **优先高质**: 优先认领高优先级客户
5. **避免囤积**: 不要认领过多客户

### 数据分析

1. **定期查看**: 每周查看转化漏斗数据
2. **对比分析**: 对比不同时间段的数据
3. **找出问题**: 识别转化率低的环节
4. **持续优化**: 根据数据优化销售策略
5. **团队分享**: 分享成功经验和数据洞察
