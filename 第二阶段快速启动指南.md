# 第二阶段快速启动指南

## 🚀 立即开始

### 1. 启动后端（如果未运行）
```powershell
cd d:/tax-crm-system
npm start
```

等待看到：
```
Server is running on port 3000
Database connected successfully
```

### 2. 启动前端（如果未运行）
```powershell
cd d:/tax-crm-system/client
npm run dev
```

等待看到：
```
VITE v5.x.x  ready in xxx ms
➜  Local:   http://localhost:5173/
```

### 3. 访问新功能

打开浏览器访问：http://localhost:5173

登录后，在左侧菜单可以看到：
- 🎯 **销售漏斗** (带"新"标记)
- ⭐ **客户分级** (带"新"标记)
- 🏷️ **标签管理** (带"新"标记)

---

## 📋 功能演示流程

### 演示1: 销售漏斗看板

1. 点击左侧菜单"销售漏斗"
2. 查看6个销售阶段（线索→意向→报价→谈判→成交→流失）
3. 点击"新增商机"按钮
4. 填写商机信息：
   - 选择客户
   - 选择阶段（如"线索"）
   - 输入标题："测试商机"
   - 输入预期金额：50000
   - 设置成交概率：60%
5. 点击"创建"
6. 看到商机卡片出现在"线索"阶段
7. **拖拽商机卡片**到"意向"阶段
8. 观察阶段自动更新

### 演示2: 标签管理

1. 点击左侧菜单"标签管理"
2. 查看8个系统标签（重点客户、潜力客户等）
3. 点击"新增标签"
4. 创建自定义标签：
   - 名称："大客户"
   - 颜色：选择红色 (#f5222d)
   - 分类："客户价值"
   - 描述："年营收超过100万的客户"
5. 点击"保存"
6. 查看标签列表中的新标签

### 演示3: 客户分级

1. 点击左侧菜单"客户分级"
2. 查看等级分布（A/B/C/D）
3. 点击"批量计算评分"按钮
4. 确认批量计算
5. 等待计算完成
6. 查看更新后的等级分布
7. 切换到"A级客户"标签
8. 查看高价值客户列表
9. 点击某个客户的"查看详情"
10. 查看评分详情（跟进评分、交易评分等）

## ⚠️ 常见问题

### Q1: 前端页面空白
**解决方案**：
1. 检查浏览器控制台（F12）是否有错误
2. 确认后端服务正在运行
3. 清除浏览器缓存（Ctrl+Shift+R）

### Q2: 拖拽不工作
**解决方案**：
1. 确认已安装 @dnd-kit 依赖
2. 刷新页面重试
3. 检查浏览器控制台错误

### Q3: 标签管理页面报错
**解决方案**：
1. 确认数据库迁移已完成
2. 检查 customer_tags 表是否存在
3. 查看后端终端错误信息

### Q4: 客户分级显示0
**解决方案**：
1. 点击"批量计算评分"按钮
2. 等待计算完成
3. 刷新页面查看结果

## 📊 数据准备

如果数据库中客户较少，建议：

1. **导入测试客户**
   - 使用客户管理页面的"导入"功能
   - 或手动创建10-20个测试客户

2. **创建测试商机**
   - 在销售漏斗页面创建5-10个商机
   - 分布在不同阶段

3. **计算评分**
   - 在客户分级页面点击"批量计算评分"
   - 等待计算完成
