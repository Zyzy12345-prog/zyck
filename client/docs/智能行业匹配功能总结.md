## 🚀 使用示例

### 在表单中使用级联选择器
```jsx
import { IndustryCascader } from '../../components';

<Form.Item label="行业" name="industryId">
  <IndustryCascader placeholder="请选择行业" />
</Form.Item>
```

### 在表格中显示匹配结果
```jsx
import { IndustryMatchDisplay } from '../../components';

{
  title: '行业',
  key: 'industry',
  render: (_, record) => (
    <IndustryMatchDisplay
      matchedIndustry={record.industryName}
      matchType={record.matchType}
      confidence={record.matchConfidence}
      compact={true}
    />
  )
}
```

### 手动选择行业
```jsx
import { IndustrySelector } from '../../components';

const [visible, setVisible] = useState(false);

<Button onClick={() => setVisible(true)}>选择行业</Button>

<IndustrySelector
  visible={visible}
  onClose={() => setVisible(false)}
  onSelect={(industry) => {
    console.log('选择了：', industry);
    setVisible(false);
  }}
/>
```

## 🔧 配置说明

### 置信度阈值
默认阈值：`0.7` (70%)

可以根据需要调整：
```javascript
// 更严格的匹配
await industryAPI.matchIndustry(text, 0.9);

// 更宽松的匹配
await industryAPI.matchIndustry(text, 0.5);
```

### 匹配策略优先级
1. 完全匹配 (exact) - 100%
2. 关键词匹配 (keyword) - 根据关键词权重
3. 同义词匹配 (synonym) - 根据同义词相似度
4. 模糊匹配 (fuzzy) - 根据字符串相似度
