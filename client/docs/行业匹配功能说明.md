# 智能行业匹配功能使用说明

## 📋 功能概述

系统提供了智能行业匹配功能，可以自动将用户输入的行业文本匹配到标准的行业分类体系中。

## 🎯 核心组件

### 1. IndustryCascader - 行业级联选择器

**功能：** 三级行业分类选择

**使用示例：**
```jsx
import { IndustryCascader } from '../../components';

<IndustryCascader
  value={industryId}
  onChange={(id) => setIndustryId(id)}
  placeholder="请选择行业"
  style={{ width: '100%' }}
/>
```

**特性：**
- ✅ 三级级联选择（大类 → 中类 → 小类）
- ✅ 支持搜索过滤
- ✅ 悬停展开子菜单
- ✅ 自动加载行业树形数据

---

### 2. IndustryMatchDisplay - 行业匹配结果显示

**功能：** 展示智能匹配结果，包含原始输入、匹配结果、匹配类型和置信度

**使用示例：**
```jsx
import { IndustryMatchDisplay } from '../../components';

<IndustryMatchDisplay
  originalText="软件开发"
  matchedIndustry="软件和信息技术服务业"
  matchType="keyword"
  confidence={0.95}
  onEdit={() => handleEdit()}
  compact={false}
/>
```

**参数说明：**
- `originalText`: 用户原始输入
- `matchedIndustry`: 匹配到的标准行业名称
- `matchType`: 匹配类型（exact/keyword/synonym/fuzzy/manual）
- `confidence`: 置信度（0-1）
- `onEdit`: 点击编辑按钮的回调
- `compact`: 是否使用紧凑模式（默认false）

**匹配类型说明：**
- 🟢 **完全匹配 (exact)**: 行业名称完全一致
- 🔵 **关键词匹配 (keyword)**: 基于预定义关键词匹配
- 🔵 **同义词匹配 (synonym)**: 使用同义词词典匹配
- 🟠 **模糊匹配 (fuzzy)**: 基于相似度算法匹配
- 🟣 **手动选择 (manual)**: 用户手动修正

**置信度颜色：**
- 🟢 90%+ : 绿色（高置信度）
- 🔵 70-89%: 蓝色（中等置信度）
- 🟠 50-69%: 橙色（低置信度）
- 🔴 <50%  : 红色（极低置信度）

---

### 3. IndustrySelector - 行业选择器模态框

**功能：** 提供搜索、树形选择和添加到字典的完整功能

**使用示例：**
```jsx
import { IndustrySelector } from '../../components';

<IndustrySelector
  visible={selectorVisible}
  onClose={() => setSelectorVisible(false)}
  onSelect={(industry) => handleSelect(industry)}
  currentIndustry={{ id: 123 }}
  originalText="软件开发"
  isAdmin={true}
/>
```

**参数说明：**
- `visible`: 是否显示模态框
- `onClose`: 关闭回调
- `onSelect`: 选择行业回调，返回 `{ id, name, code, level, matchType }`
- `currentIndustry`: 当前选中的行业（可选）
- `originalText`: 原始输入文本（用于添加到字典）
- `isAdmin`: 是否显示管理员功能（添加到字典）

**特性：**
- ✅ 实时搜索过滤
- ✅ 树形结构展示
- ✅ 高亮搜索关键词
- ✅ 管理员可添加关键词到字典

## 🚀 在客户导入中的应用

1. **上传Excel文件**
   - 支持 .xlsx 和 .xls 格式

## 🔧 API接口

### 获取行业树形结构
```javascript
GET /api/clients/industries/tree
```

### 获取行业列表（扁平）
```javascript
GET /api/clients/industries/list?level=1
```

### 单个行业匹配
```javascript
POST /api/clients/industries/match
{
  "text": "软件开发",
  "threshold": 0.7
}
```

### 批量行业匹配
```javascript
POST /api/clients/industries/batch-match
{
  "texts": ["软件开发", "金融服务", "制造业"],
  "threshold": 0.7
}
```

### 更新行业关键词（管理员）
```javascript
PUT /api/clients/industries/:id/keywords
{
  "keywords": ["软件", "开发", "IT"]
}
```

## 💡 最佳实践

### 1. 导入前准备
- 下载并使用官方模板
- 确保行业名称尽量规范
- 必填字段不能为空

### 2. 匹配结果处理
- 置信度 ≥ 90%：通常可以直接使用
- 置信度 70-89%：建议人工复核
- 置信度 < 70%：建议手动修正

### 3. 提高匹配准确率
- 使用标准行业名称
- 避免使用缩写或俗称
- 管理员可将常用别名添加到字典

### 4. 字典管理（管理员）
- 定期审查未匹配的行业
- 将高频词汇添加到对应行业的关键词
- 维护同义词和别名映射

## 🐛 常见问题

### Q: 为什么有些行业匹配不到？
A: 可能原因：
1. 输入的行业名称不在标准分类中
2. 使用了非标准的缩写或别名
3. 置信度阈值设置过高

**解决方案：**
- 手动选择正确的行业
- 管理员将该名称添加到字典

### Q: 如何提高匹配准确率？
A: 
1. 使用完整的行业名称
2. 参考国家统计局的行业分类标准
3. 管理员维护关键词字典

### Q: 批量导入时部分失败怎么办？
A: 
1. 查看导入结果中的错误信息
2. 修正Excel中的错误数据
3. 重新导入失败的记录
