## 🌐 API接口检查

### 在浏览器控制台执行以下测试：

```javascript
const token = localStorage.getItem('token');

// 1. 测试行业列表API
fetch('http://localhost:3000/api/clients/industries/list', {
  headers: { 'Authorization': `Bearer ${token}` }
})
.then(res => res.json())
.then(data => console.log('✅ 行业列表:', data.data?.length, '条'))
.catch(err => console.error('❌ 失败:', err));

// 2. 测试行业树API
fetch('http://localhost:3000/api/clients/industries/tree', {
  headers: { 'Authorization': `Bearer ${token}` }
})
.then(res => res.json())
.then(data => console.log('✅ 行业树:', data.data?.length, '个大类'))
.catch(err => console.error('❌ 失败:', err));

// 3. 测试单个匹配API
fetch('http://localhost:3000/api/clients/industries/match', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ text: '软件开发', threshold: 0.7 })
})
.then(res => res.json())
.then(data => {
  if (data.success && data.data.matched) {
    console.log('✅ 匹配成功:', data.data.matchedIndustry.name);
  } else {
    console.log('⚠️ 未匹配');
  }
})
.catch(err => console.error('❌ 失败:', err));

// 4. 测试批量匹配API
fetch('http://localhost:3000/api/clients/industries/batch-match', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    texts: ['软件开发', '金融服务', '制造业'],
    threshold: 0.7
  })
})
.then(res => res.json())
.then(data => {
  console.log('✅ 批量匹配:', data.data.statistics);
})
.catch(err => console.error('❌ 失败:', err));
```

**预期结果：**
- ✅ 行业列表: 42 条
- ✅ 行业树: 19 个大类
- ✅ 匹配成功: 软件和信息技术服务业
- ✅ 批量匹配: {matched: 3, unmatched: 0, total: 3}
