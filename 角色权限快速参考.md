## 🔑 权限代码格式

```
格式：module:action

示例：
  client:read      - 查看客户
  client:create    - 创建客户
  call:update      - 编辑外呼
  employee:delete  - 删除员工
  *                - 所有权限
```

## 💻 常用代码

### 创建角色
```javascript
const role = await Role.create({
  name: '区域经理',
  code: 'REGIONAL_MANAGER',
  description: '管理区域销售',
  permissions: ['client:read', 'client:update'],
  level: 65,
  isActive: true
});
```

### 查找角色
```javascript
// 按代码查找
const role = await Role.findByCode('ADMIN');

// 按名称查找
const role = await Role.findByName('系统管理员');

// 获取启用的角色
const roles = await Role.getActiveRoles();
```

### 角色权限操作
```javascript
// 检查权限
if (role.hasPermission('client:read')) { }

// 添加权限
await role.addPermissions(['client:write']);

// 移除权限
await role.removePermissions(['client:delete']);

// 同步权限
await role.syncPermissions();
```

### 创建权限
```javascript
const permission = await Permission.create({
  code: 'invoice:create',
  name: '创建发票',
  module: 'invoice',
  action: 'create',
  description: '创建新发票',
  group: '发票管理',
  sortOrder: 100
});
```

### 查找权限
```javascript
// 按代码查找
const perm = await Permission.findByCode('client:read');

// 按模块查找
const perms = await Permission.findByModule('client');

// 按模块分组
const grouped = await Permission.getGroupedByModule();
```

### 权限检查中间件
```javascript
const checkPermission = (requiredPerm) => {
  return async (req, res, next) => {
    const employee = await Employee.findByPk(req.user.id, {
      include: [{ model: Role, as: 'role' }]
    });

    if (!employee?.role?.hasPermission(requiredPerm)) {
      return res.status(403).json({ error: '权限不足' });
    }

    next();
  };
};

// 使用
router.post('/clients', 
  authenticate, 
  checkPermission('client:create'), 
  createClient
);
```

### 查询角色及权限
```javascript
const role = await Role.findByPk(1, {
  include: [{
    model: Permission,
    as: 'permissionDetails',
    through: { attributes: [] }
  }]
});
```

### 员工权限检查
```javascript
const employee = await Employee.findByPk(1, {
  include: [{
    model: Role,
    as: 'role',
    include: [{
      model: Permission,
      as: 'permissionDetails'
    }]
  }]
});

// 检查权限
if (employee.role?.hasPermission('client:write')) {
  // 允许操作
}
```

## ⚠️ 注意事项

1. **系统角色** - `isSystem: true` 的角色不可删除
2. **权限代码** - 格式必须是 `module:action`
3. **角色等级** - 1-100，数字越大权限越高
4. **超级权限** - `*` 代表所有权限
5. **数据一致性** - 删除角色时员工roleId设为NULL

## 🎯 权限检查流程

```
请求 → 身份验证 → 获取角色 → 检查权限 → 允许/拒绝
```

### 检查优先级
1. 超级权限 (`*`)
2. 角色权限 (role.permissions)
3. 员工个人权限 (employee.permissions)
4. 关联表权限 (role_permissions)
