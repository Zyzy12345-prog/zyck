## 🏗️ 技术架构

### 前端技术栈
```
React 18.3.1          # UI框架
Ant Design 5.x        # UI组件库
React Router 7.12.0   # 路由管理
Axios 1.13.2          # HTTP客户端
XLSX 0.18.5           # Excel处理
dayjs                 # 日期处理
```

### 项目结构
```
client/src/
├── pages/
│   └── employee/
│       ├── EmployeeList.jsx       # 员工列表
│       ├── EmployeeList.css
│       ├── EmployeeForm.jsx       # 员工表单
│       ├── EmployeeForm.css
│       ├── EmployeeImport.jsx     # 批量导入
│       ├── EmployeeImport.css
│       ├── EmployeeDetail.jsx     # 员工详情（待开发）
│       └── EmployeeDetail.css
├── services/
│   └── api.js                     # API服务
├── components/
│   └── Layout.jsx                 # 布局组件
└── App.jsx                        # 路由配置
```

### API接口
```javascript
// 员工相关API
employeeAPI.getEmployees(params)      // 获取员工列表
employeeAPI.getEmployee(id)           // 获取员工详情
employeeAPI.createEmployee(data)      // 创建员工
employeeAPI.updateEmployee(id, data)  // 更新员工
employeeAPI.deleteEmployee(id)        // 删除员工
employeeAPI.updateEmployeeStatus(id, status)  // 修改状态
employeeAPI.resetPassword(id)         // 重置密码

// 角色相关API
roleAPI.getRoles(params)              // 获取角色列表
```

## 🚀 部署说明

### 环境要求
```
Node.js: >=16.0.0
npm: >=8.0.0
浏览器: Chrome/Firefox/Safari/Edge (最新版)
```

### 安装步骤
```bash
# 1. 克隆项目
git clone <repository-url>

# 2. 安装依赖
cd tax-crm-system/client
npm install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，设置 VITE_API_URL

# 4. 启动开发服务器
npm run dev

# 5. 构建生产版本
npm run build
```

### 配置说明
```javascript
// .env 文件
VITE_API_URL=http://localhost:3000/api  # API地址
```

### 路由配置
```javascript
// App.jsx
<Route path="/employees" element={<EmployeeList />} />
<Route path="/employees/create" element={<EmployeeForm />} />
<Route path="/employees/:id/edit" element={<EmployeeForm />} />
<Route path="/employees/:id" element={<EmployeeDetail />} />
```
