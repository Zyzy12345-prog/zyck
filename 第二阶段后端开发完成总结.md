## 📊 API接口清单

### 销售漏斗 API (15个)
```
GET    /api/sales-funnel/stages                    获取销售阶段
GET    /api/sales-funnel/opportunities             获取商机列表
GET    /api/sales-funnel/opportunities/by-stage    按阶段获取商机(看板)
GET    /api/sales-funnel/opportunities/:id         获取单个商机
POST   /api/sales-funnel/opportunities             创建商机
PUT    /api/sales-funnel/opportunities/:id         更新商机
DELETE /api/sales-funnel/opportunities/:id         删除商机
PUT    /api/sales-funnel/opportunities/:id/move-stage  移动阶段(拖拽)
PUT    /api/sales-funnel/opportunities/:id/mark-won    标记成交
PUT    /api/sales-funnel/opportunities/:id/mark-lost   标记流失
GET    /api/sales-funnel/statistics                漏斗统计
```

### 标签管理 API (11个)
```
GET    /api/customer-tags/tags                     获取所有标签
GET    /api/customer-tags/tags/categories          获取标签分类
GET    /api/customer-tags/tags/statistics          标签使用统计
POST   /api/customer-tags/tags                     创建标签
PUT    /api/customer-tags/tags/:id                 更新标签
DELETE /api/customer-tags/tags/:id                 删除标签
POST   /api/customer-tags/client-tags              添加标签到客户
POST   /api/customer-tags/client-tags/batch        批量添加标签
DELETE /api/customer-tags/client-tags/:clientId/:tagId  移除标签
GET    /api/customer-tags/clients/:clientId/tags   获取客户标签
GET    /api/customer-tags/tags/:tagId/clients      获取标签下的客户
```

### 客户评分 API (8个)
```
POST   /api/client-scoring/clients/:clientId/calculate-score      计算评分
POST   /api/client-scoring/clients/batch-calculate-scores         批量计算
GET    /api/client-scoring/clients/:clientId/score                获取评分
POST   /api/client-scoring/clients/:clientId/update-value-analysis 更新价值分析
GET    /api/client-scoring/clients/:clientId/value-analysis       获取价值分析
GET    /api/client-scoring/clients/level/:level                   按等级查询
GET    /api/client-scoring/clients/level-distribution             等级分布
GET    /api/client-scoring/clients/high-value                     高价值客户
GET    /api/client-scoring/clients/at-risk                        流失风险客户
```
