## 🚀 如何启动

### 方法1：使用启动脚本（最简单）

**PowerShell:**
```powershell
cd d:/tax-crm-system
./start.ps1
```

**CMD:**
```cmd
cd d:\tax-crm-system
start.bat
```

然后选择"3. 同时启动前后端"

### 方法2：手动启动

**终端1 - 后端:**
```powershell
cd d:/tax-crm-system
npm run dev
```

**终端2 - 前端:**
```powershell
cd d:/tax-crm-system/client
npm run dev
```

### 访问地址
- 🌐 前端: http://localhost:5173
- 🔌 后端: http://localhost:3000
