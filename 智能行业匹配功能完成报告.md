### 1. 后端智能匹配服务
**文件：** `services/IndustryMatchingService.js`

**核心功能：**
- ✅ **完全匹配**：精确匹配行业名称（置信度 1.0）
- ✅ **关键词匹配**：基于预定义关键词库（置信度 0.95）

- ✅ **批量处理**：支持批量匹配多个行业
- ✅ **智能建议**：未匹配时提供相似行业建议

## 🔧 配置说明

### 置信度阈值
默认阈值：**0.7**（70%）

- **≥ 0.9**：高置信度（完全匹配、关键词匹配）
- **0.7 - 0.9**：中等置信度（同义词匹配、模糊匹配）
- **< 0.7**：低置信度（不采用，提供建议）

### 调整阈值
```javascript
// 更严格的匹配（只接受高置信度）
await industryAPI.matchIndustry("科研", 0.85);

// 更宽松的匹配（接受更多模糊匹配）
await industryAPI.matchIndustry("科研", 0.6);
```

管理员可以为行业添加新的关键词，改进匹配效果：

```javascript
// 为"科学研究和技术服务业"添加关键词
await industryAPI.updateIndustryKeywords(industryId, [
  '科研',
  '技术研发',
  'R&D',
  '研究开发',
  '科学技术',
  '技术服务',
  '科技创新',
  '实验室'
]);
```

### 2. 匹配统计
批量匹配会返回详细统计信息：

```javascript
{
  total: 100,           // 总数
  matched: 85,          // 成功匹配
  unmatched: 15,        // 未匹配
  byMatchType: {
    exact: 30,          // 完全匹配
    keyword: 40,        // 关键词匹配
    synonym: 10,        // 同义词匹配
    fuzzy: 5            // 模糊匹配
  },
  averageConfidence: 0.87,  // 平均置信度
  lowConfidence: [...]      // 低置信度列表
}
```

### 3. 未匹配时的建议
当无法匹配时，系统会提供相似行业建议：

```javascript
{
  matched: false,
  matchedIndustry: null,
  suggestions: [
    { name: "科学研究和技术服务业", confidence: 0.65 },
    { name: "信息传输、软件和信息技术服务业", confidence: 0.58 },
    // ...
  ]
}
```
