========================================
  🚀 如何启动
========================================

方法1：使用启动脚本（推荐）
  
  PowerShell:
    cd d:/tax-crm-system
    ./start.ps1
  
  CMD:
    cd d:\tax-crm-system
    start.bat
  
  然后选择 "3. 同时启动前后端"

方法2：手动启动

  终端1 - 启动后端:
    cd d:/tax-crm-system
    npm run dev
  
  终端2 - 启动前端:
    cd d:/tax-crm-system/client
    npm run dev
