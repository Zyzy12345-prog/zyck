## 💻 常用代码片段

### 1. 获取所有一级分类
```javascript
const categories = await IndustryCategory.findAll({
  where: { level: 1, isActive: true },
  order: [['sortOrder', 'ASC']]
});
```

### 2. 获取树形结构
```javascript
const tree = await IndustryCategory.getTree();
```

### 3. 智能匹配行业
```javascript
const matched = await IndustryCategory.matchByKeywords('软件开发');
const path = await matched.getFullPath();
// 返回: "科技与互联网 > 信息技术服务"
```

### 4. 查询客户及行业
```javascript
const client = await Client.findByPk(id, {
  include: [{
    model: IndustryCategory,
    as: 'industry'
  }]
});
```

### 5. 创建客户并关联行业
```javascript
const client = await Client.create({
  companyName: '某某公司',
  originalIndustry: '软件开发',
  industryId: 101,
  // ...
});
```

## 🔍 SQL查询示例

### 查看所有行业
```sql
SELECT 
  id, 
  parent_id, 
  name, 
  code, 
  level 
FROM industry_categories 
ORDER BY level, sort_order;
```

### 查看行业树形结构
```sql
SELECT 
  p.name as parent_name,
  c.name as child_name,
  c.code
FROM industry_categories p
LEFT JOIN industry_categories c ON c.parent_id = p.id
WHERE p.level = 1
ORDER BY p.sort_order, c.sort_order;
```

### 统计各行业客户数
```sql
SELECT 
  ic.name,
  COUNT(c.id) as client_count
FROM industry_categories ic
LEFT JOIN clients c ON c.industry_id = ic.id
GROUP BY ic.id, ic.name
ORDER BY client_count DESC;
```

## 📋 数据库字段说明

### industry_categories 表
- `id`: 主键
- `parent_id`: 父级ID（0=一级分类）
- `name`: 行业名称
- `code`: 行业代码（唯一）
- `level`: 层级（1/2/3）
- `is_active`: 是否启用
- `keywords`: 关键词JSON数组
- `sort_order`: 排序顺序

### clients 表（新增字段）
- `original_industry`: 原始行业文本
- `industry_id`: 行业分类ID（外键）

## 🆘 常见问题

### Q: 如何添加新的行业分类？
```javascript
await IndustryCategory.create({
  parentId: 1,  // 父级ID
  name: '新行业',
  code: 'NEW_CODE',
  level: 2,
  keywords: ['关键词1', '关键词2'],
  sortOrder: 10
});
```

### Q: 如何批量更新客户的行业分类？
```javascript
const clients = await Client.findAll({
  where: { industryId: null }
});

for (const client of clients) {
  if (client.originalIndustry) {
    const matched = await IndustryCategory.matchByKeywords(
      client.originalIndustry
    );
    if (matched) {
      await client.update({ industryId: matched.id });
    }
  }
}
```

### Q: 如何禁用某个行业分类？
```javascript
await IndustryCategory.update(
  { isActive: false },
  { where: { id: 101 } }
);
```
