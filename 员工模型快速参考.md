# 员工模型快速参考

## 🎯 核心功能速查

### 创建员工
```javascript
const employee = await Employee.create({
  employeeId: 'EMP001',
  username: 'john_doe',
  password: 'password123',
  fullName: '张三',
  email: 'zhangsan@example.com',
  phone: '13800138000',
  department: '销售部',
  position: '销售经理',
  hireDate: '2024-01-15',
  roleId: 2,
  permissions: ['client:read', 'client:write'],
  accessLevel: 5
});
```

### 员工登录验证
```javascript
const employee = await Employee.findByUsername('john_doe');
const isValid = await employee.comparePassword('password123');
if (isValid && employee.isActive()) {
  await employee.updateLastLogin('192.168.1.100');
  return employee.toJSON(); // 自动隐藏密码
}
```

### 权限管理
```javascript
// 检查权限
if (employee.hasPermission('client:write')) { /* ... */ }

// 添加权限
await employee.addPermissions(['report:view', 'report:export']);

// 移除权限
await employee.removePermissions(['report:export']);
```

### 更新统计
```javascript
await employee.updateStats({
  totalClients: 100,
  activeClients: 60,
  completedCalls: 500,
  successRate: 80.5
});
```

### 常用查询
```javascript
// 根据用户名查找
const emp = await Employee.findByUsername('john_doe');

// 根据工号查找
const emp = await Employee.findByEmployeeId('EMP001');

// 根据邮箱查找
const emp = await Employee.findByEmail('john@example.com');

// 获取在职员工
const active = await Employee.getActiveEmployees();

// 按部门查询
const sales = await Employee.getByDepartment('销售部');
```

## 📊 字段速查表

| 字段 | 类型 | 必填 | 唯一 | 说明 |
|------|------|------|------|------|
| employeeId | STRING(20) | ✅ | ✅ | 员工工号 |
| username | STRING(50) | ✅ | ✅ | 登录用户名 |
| password | STRING(255) | ✅ | ❌ | 密码（加密） |
| fullName | STRING(100) | ✅ | ❌ | 姓名 |
| email | STRING(100) | ✅ | ✅ | 邮箱 |
| phone | STRING(20) | ❌ | ❌ | 电话 |
| department | STRING(100) | ❌ | ❌ | 部门 |
| position | STRING(100) | ❌ | ❌ | 职位 |
| status | ENUM | ✅ | ❌ | active/inactive/suspended |
| roleId | INTEGER | ❌ | ❌ | 角色ID |
| permissions | ARRAY | ❌ | ❌ | 权限数组 |
| accessLevel | INTEGER | ✅ | ❌ | 访问等级(1-10) |
| totalClients | INTEGER | ✅ | ❌ | 客户总数 |
| activeClients | INTEGER | ✅ | ❌ | 活跃客户数 |
| completedCalls | INTEGER | ✅ | ❌ | 完成通话数 |
| successRate | DECIMAL(5,2) | ✅ | ❌ | 成功率(0-100) |
