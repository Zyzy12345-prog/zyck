# 员工管理模块 - README

## 📖 简介

税务CRM系统的员工管理模块前端界面，提供完整的员工信息管理、批量导入、权限管理等功能。

## ✨ 核心功能

- 🔍 **员工列表**: 搜索、筛选、分页、统计
- ✏️ **员工管理**: 新增、编辑、删除、状态管理
- 📥 **批量导入**: Excel导入、数据验证、结果导出
- 🔐 **权限管理**: 角色分配、管理员权限
- 📱 **响应式设计**: 支持桌面、平板、手机

## 🚀 快速开始

```bash
# 安装依赖
cd client
npm install

# 启动开发服务器
npm run dev

# 访问地址
http://localhost:5173/employees
```

## 🎯 主要特性

### 1. 员工列表
- 数据表格展示（12列）
- 全局搜索（姓名、工号、邮箱、手机号）
- 多维度筛选（部门、职位、状态、角色）
- 分页功能（10/20/50/100条/页）
- 统计卡片（总数、在职、停用、离职）

### 2. 员工表单
- 创建/编辑模式
- 头像上传（JPG/PNG，最大2MB）
- 15个基本信息字段
- 前端验证 + 异步验证
- 自动生成工号和密码

### 3. 批量导入
- 四步导入流程
- Excel模板下载
- 数据验证（15项规则）
- 实时进度显示
- 结果导出（含初始密码）
