## 🗄️ 数据库结构

### 1. 行业分类表 (industry_categories)

```sql
CREATE TABLE industry_categories (
  id SERIAL PRIMARY KEY,
  parent_id INTEGER NOT NULL DEFAULT 0,  -- 父级ID，0表示一级分类
  name VARCHAR(100) NOT NULL,            -- 行业名称
  code VARCHAR(50),                      -- 行业代码
  level INTEGER NOT NULL DEFAULT 1,      -- 层级：1/2/3
  is_active BOOLEAN NOT NULL DEFAULT true, -- 是否启用
  keywords JSON,                         -- 关键词数组（智能匹配）
  sort_order INTEGER NOT NULL DEFAULT 0, -- 排序顺序
  created_at TIMESTAMP WITH TIME ZONE NOT NULL,
  updated_at TIMESTAMP WITH TIME ZONE NOT NULL
);

-- 索引
CREATE INDEX idx_industry_parent_id ON industry_categories(parent_id);
CREATE INDEX idx_industry_level ON industry_categories(level);
CREATE INDEX idx_industry_is_active ON industry_categories(is_active);
```

### 2. 客户表更新 (clients)

**字段变更**:
- `industry` → `original_industry` (重命名，保留原始文本)
- 新增 `industry_id` (外键关联 industry_categories)

```sql
ALTER TABLE clients 
  RENAME COLUMN industry TO original_industry;

ALTER TABLE clients 
  ADD COLUMN industry_id INTEGER 
  REFERENCES industry_categories(id) 
  ON DELETE SET NULL 
  ON UPDATE CASCADE;

CREATE INDEX idx_clients_industry_id ON clients(industry_id);
```

## 🚀 使用示例

### 1. 查询所有一级分类
```javascript
const level1 = await IndustryCategory.findAll({
  where: { level: 1, isActive: true },
  order: [['sortOrder', 'ASC']]
});
```

### 2. 查询某个分类的子分类
```javascript
const children = await IndustryCategory.findAll({
  where: { parentId: 8, isActive: true },
  order: [['sortOrder', 'ASC']]
});
```

### 3. 获取树形结构
```javascript
const tree = await IndustryCategory.getTree();
```

### 4. 智能匹配行业
```javascript
const matched = await IndustryCategory.matchByKeywords('税务代理服务');
if (matched) {
  const fullPath = await matched.getFullPath();
  console.log(fullPath); // "咨询服务 > 税务咨询"
}
```

### 5. 创建客户时关联行业
```javascript
const client = await Client.create({
  companyName: '某某科技有限公司',
  originalIndustry: '软件开发',
  industryId: 101,  // 信息技术服务
  // ... 其他字段
});
```

### 6. 查询客户及其行业信息
```javascript
const client = await Client.findByPk(1, {
  include: [{
    model: IndustryCategory,
    as: 'industry',
    include: [{
      model: IndustryCategory,
      as: 'parent'
    }]
  }]
});

console.log(client.industry.name);        // "信息技术服务"
console.log(client.industry.parent.name); // "科技与互联网"
```

-- 查看客户表结构
\d clients

SELECT 
  ic.name,
  COUNT(c.id) as client_count
FROM industry_categories ic
LEFT JOIN clients c ON c.industry_id = ic.id
GROUP BY ic.id, ic.name
ORDER BY client_count DESC;
