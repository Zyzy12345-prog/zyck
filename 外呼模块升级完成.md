## 🚀 如何使用新功能

### 1. 启动系统

```powershell
# 终端1：启动后端
cd d:/tax-crm-system
npm run dev

# 终端2：启动前端
cd d:/tax-crm-system/client
npm run dev
```

### 2. 访问外呼模块

浏览器打开：http://localhost:5173/calls

### 3. 创建外呼记录

1. 点击"新增记录"按钮
2. 填写表单：
   - **选择客户**（必填）- 电话号码会自动填充
   - **电话号码**（必填）- 可手动修改
   - **呼叫类型** - 选择呼出/呼入
   - **呼叫方向** - 选择方向
   - **呼叫时间**（必填）- 选择日期时间
   - **通话时长** - 输入分钟和秒
   - **呼叫状态**（必填）- 选择7种状态之一
   - **通话摘要** - 简要描述
   - **备注信息** - 详细记录
   - **标签分类** - 输入标签后按回车添加
   - **满意度评分** - 选择1-5星
   - **后续行动计划** - 输入下一步行动
   - **下次跟进时间** - 选择日期

3. 点击"创建"按钮

### 4. 编辑外呼记录

1. 在列表中点击记录的"编辑"按钮
2. 修改任意字段
3. 点击"保存"按钮

## 📈 性能优化

### 数据库索引
已创建10个索引提升查询性能：

1. `idx_call_records_client_id` - 按客户查询
2. `idx_call_records_user_id` - 按用户查询
3. `idx_call_records_call_time` - 按时间查询
4. `idx_call_records_status` - 按状态查询
5. `idx_call_records_direction` - 按方向查询
6. `idx_call_records_phone` - 按电话查询
7. `idx_call_records_follow_up` - 按跟进日期查询
8. `idx_call_records_satisfaction` - 按满意度查询
9. `idx_call_records_client_user` - 复合索引（客户+用户）
10. `idx_call_records_time_status` - 复合索引（时间+状态）
