## 🛠️ 技术实现

### 数据库设计

**industry_categories 表结构**:
```sql
- id: 主键
- parent_id: 父级ID（0表示一级分类）
- name: 行业名称
- code: 行业代码（唯一）
- level: 层级（1/2/3）
- is_active: 是否启用
- keywords: 关键词JSON数组
- sort_order: 排序顺序
- created_at: 创建时间
- updated_at: 更新时间
```

**clients 表更新**:
```sql
- original_industry: 原始行业文本（从industry重命名）
- industry_id: 行业分类ID（新增外键）
```

### 代码示例

**查询所有一级分类**:
```javascript
const categories = await IndustryCategory.findAll({
  where: { level: 1, isActive: true },
  order: [['sortOrder', 'ASC']]
});
```

**智能匹配行业**:
```javascript
const matched = await IndustryCategory.matchByKeywords('软件开发');
const path = await matched.getFullPath();
// 返回: "科技与互联网 > 信息技术服务"
```

```javascript
const client = await Client.create({
```
