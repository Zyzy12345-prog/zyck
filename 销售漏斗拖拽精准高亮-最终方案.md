## 🔧 技术细节

### closestCenter 算法

**计算公式：**
```javascript
// 计算两点之间的欧几里得距离
distance = √[(x2-x1)² + (y2-y1)²]

// 对每个阶段计算距离
distances = [
  { id: 1, distance: 50 },   // 线索
  { id: 2, distance: 150 },  // 意向
  { id: 3, distance: 250 },  // 报价
  ...
]

// 返回距离最小的阶段
return distances.sort((a, b) => a.distance - b.distance)[0]
```

**优势：**
- ✅ 数学上最精准
- ✅ 考虑了 X 和 Y 两个维度
- ✅ 适用于任何布局
