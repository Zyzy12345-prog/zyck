## 🔌 API 接口

### 员工管理接口（9个）
1. `GET /api/employees` - 获取员工列表
2. `GET /api/employees/statistics` - 获取统计数据
3. `GET /api/employees/:id` - 获取员工详情
4. `POST /api/employees` - 创建员工
5. `PUT /api/employees/:id` - 更新员工
6. `DELETE /api/employees/:id` - 删除员工
7. `PATCH /api/employees/:id/status` - 修改状态
8. `POST /api/employees/:id/reset-password` - 重置密码
9. `POST /api/employees/:id/change-password` - 修改密码

### 辅助接口（4个）
1. `GET /api/employees/options/departments` - 获取部门列表
2. `GET /api/employees/options/positions` - 获取职位列表
3. `GET /api/employees/options/roles` - 获取角色列表
4. `POST /api/employees/import` - 批量导入

**总计：13个API接口**
